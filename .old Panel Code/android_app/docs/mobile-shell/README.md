# Mobile PWA App Shell Docs

This folder contains execution documents for shipping a native-distributed shell app (Android + iOS) that loads the existing mobile PWA.

## Files
- PHASE1_PRODUCT_AND_COMPLIANCE_PLAN.md
- PHASE1_COMPLIANCE_CHECKLIST.md
- PHASE1_EXECUTION_BOARD.md
- PHASE2_TO_PHASE7_ANDROID_EXECUTION.md
- ANDROID_QA_SECURITY_CHECKLIST.md
- ANDROID_RELEASE_PIPELINE.md

## How to use
1. Complete all checkboxes in PHASE1_COMPLIANCE_CHECKLIST.md.
2. Finalize approvals and decisions in PHASE1_PRODUCT_AND_COMPLIANCE_PLAN.md.
3. Track day-wise progress in PHASE1_EXECUTION_BOARD.md.
4. Start Phase 2 only after all Phase 1 acceptance criteria are complete.
